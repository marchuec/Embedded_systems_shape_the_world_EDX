UT.6.10x Embedded Systems - Shape The World: Microcontroller Input/Output

and

UT.6.20x Embedded Systems - Shape The World: Multi-Threaded Interfacing

Test EXecute, And Simulate, called TExaS, is a collection of DLL plugins that transform Keil uVision into learning environment for embedded systems. These plugins provide simulation of the Texas Instruments Tiva LaunchPad. The suite also includes a stand-alone application called TExaSdisplay, which provides a serial port interface. When operating with the code embedded on the microcontroller, TExaSdisplay also provides a oscilloscope functionality. Furthermore, TExaS can be used for automatic grading of labs both in simulation and on the real board.
